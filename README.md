### Wiz 센터에서 진행하는 HTML5, CSS 강좌에 대한 실습  
실습 내용 중 알고 있는 부분을 제외한 잘 알지 못했거나 처음 본 내용에 대해 실습한 내용을 커밋한 레파지토리입니다.  
즉 이것도 별 거 없는 레파지토리이므로 언제 삭제될지 모릅니다.  
기존에 모든 작업은 Eclipse로 하다가 웹 관련 툴로 Visual Studio for Web도 자주 사용한다고 해서 한번 사용해봅니다.  
그러므로 이번 레파지토리의 프로젝트는 비쥬얼 스튜디오의 솔루션 형태입니다.  

#### HTML5
html에 대한 내용이 저장되는 프로젝트로, html5에 대한 내용이 주로 들어갈 것으로 예상됩니다.  
강의 내용에 따라 어찌될지는 모릅니다.  
  
#### CSS3  
간단한 HTML 문서에 CSS를 적용하는 내용일듯 합니다.
overflow : 자식 요소가 부모의 범위를 벗어날 때 어떻게 처리할 것인지 정하는 속성인듯
float: 블록 요소를 라인요소처럼 사용하기 위해 사용하는 속성인듯
clear: 상위 요소에 의해 상속받은 float 속성을 제거하는 속성인듯
